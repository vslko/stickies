Stickies demo
To add sticky, move mouse cursor to right side of page.
Features
  • Remember notes on page
  • Save text and position
  • Easy add and remove
  • Drag and Drop support
Getting started
Insert in head
<!-- basic style -->
<link rel="stylesheet" type="text/css" href="your_path/css/stickies.css" />
<!-- required javascripts -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.1/jquery.min.js"></script>
<script type="text/javascript" src="your_path/js/stickies.js"></script>
Insert in body
<script type="text/javascript">
$(document).ready( function(){
	$(this).stickies({
		url_get    : './server/get.php',
		url_add    : './server/add.php',
		url_edit   : './server/edit.php',
		url_remove : './server/remove.php',
	});
});
</script>
Documentation
Options
get_urlType: string, Default: "./" Url for request to fetch stickies from server add_urlType: string, Default: "./add" Url for request to add new sticky edit_urlType: string, Default: "./edit" Url for request to save sticky remove_urlType: string, Default: "./remove" Url for request to remove sticky paramsType: object, Default: {} Additional parameter for request to server heightType: integer, Default: 300 Z-Index for stickies, must be greater than any element on page.
